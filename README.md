# Koyeb-keepalive

使用 Github Actions 以让 Koyeb 保持账户活跃

## 部署说明

* Fork 这个项目，然后启动
* 项目用到的环境变量
  | 变量名 | 是否必须 | 默认值 | 备注 |
  | ------------ | ------ | ------ | ------ |
  | KOY_EB | 是 | 无 | 账号密码用`-`分割，多账户用&隔开[例如：aaa-bbb&ccc-ddd] |
